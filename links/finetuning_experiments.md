# Supervised Finetuning Experiments

This document provides links to the artifacts generated from the supervised
finetuning experiments. These artifacts include datasets, models, and evaluation
results that were produced during the experimentation phase.

## Supervised Finetuning Experiments Artifacts

The following link pertains to Supervised Finetuning Experiments:

[BERTology of Molecular Property Prediction (Supervised Finetuning Experiments)](https://doi.org/10.5281/zenodo.19074661)

It involves the following files:

- tiny-bpe-vocab30522-ms-1234-ds-1234.tar.gz
- tiny-bpe-vocab30522-ms-1234-ds-2345.tar.gz
- tiny-bpe-vocab30522-ms-2345-ds-1234.tar.gz
- small-bpe-vocab30522-ms-1234-ds-1234.tar.gz
- small-bpe-vocab30522-ms-1234-ds-2345.tar.gz
- small-bpe-vocab30522-ms-2345-ds-1234.tar.gz
- base-bpe-vocab30522-ms-1234-ds-1234.tar.gz
- base-bpe-vocab30522-ms-1234-ds-2345.tar.gz
- base-bpe-vocab30522-ms-2345-ds-1234.tar.gz

Each tar file contains all model artifacts (checkpoints, random-number generator
states, optimizer states etc.), training logs (Tensorboard, MLFlow and Weights &
Biases), and evaluation results, configuration files, run scripts, SLURM sbatch
driver scripts, and any additional artifacts generated during the experiments.
